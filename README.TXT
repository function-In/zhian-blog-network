搭建环境：
工具：XP（ 小皮工具箱 ）
1、在数据库导入 sql 文件
2、打开小皮，创建一个域名为 zhian.cn 的站点
3、把网站源码放进域名为 zhian.cn 的站点根目录下
4、使用 IDE 编辑源码 data 目录下的 data.php 文件，对数据库地址、数据库登录名称、数据库登录密码、选择数据库等的接入进行配置编辑
5、打开创建好的站点，输入登录用户名称和用户密码

文件夹相关说明：
ZHIAN => PHP-ZHIAN站点存放文件夹

后台管理文件夹说明：
ZHIAN -> admin => 后台管理页面文件夹
ZHIAN -> admin -> pages => 后台管理跳转单独页面存放文件夹
ZHIAN -> admin -> public => 页面共用区域存放文件夹

数据库管理文件夹：
ZHIAN -> data => 数据库调用封装函数存放文件夹，以及相关 API 接口

页面静态相关信息文件夹：
ZHIAN -> static => 存放静态文件，如images，css，js等
ZHIAN -> static -> fonts => 页面字体文件存放文件夹
ZHIAN -> static -> images => 页面图片文件存放文件夹
ZHIAN -> static -> scripts => js插件文件存放文件夹
ZHIAN -> static -> styles => css样式文件存放文件夹

上传管理文件夹：
ZHIAN -> uploads
